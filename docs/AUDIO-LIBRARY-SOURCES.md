## Required Formats for Reel Kingdom Studios

Only these formats are recognized. Other formats (e.g. .flac, .m4a in some containers) will not appear in the Assembly dropdown.

| Format | Extensions |
|--------|------------|
| MP3 | `.mp3` |
| WAV | `.wav` |
| OGG | `.ogg` |
| AAC / M4A | `.aac`, `.m4a` |
| AIFF | `.aiff`, `.aif` |

**Tip:** If you have files in other formats (e.g. .flac), convert with:
```bash
ffmpeg -i input.flac -c:a libmp3lame -q:a 2 output.mp3
```
