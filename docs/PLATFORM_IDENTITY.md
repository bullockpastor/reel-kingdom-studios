# Reel Kingdom Studios — Value Proposition

## What It Is

Reel Kingdom Studios is a **production orchestration platform** for AI video. It does not replace Sora, Veo, or Runway — it **uses them** as part of a full pipeline.

---

## The Key Distinction

| Sora / Veo / Runway | Reel Kingdom Studios |
|---------------------|-----------------------|
| Generate single clips | Produce finished videos |
| One prompt → one clip | Storyboard → orchestrate → QC → assemble → ship |
| Pay per clip, every time | Use local for drafts, premium for finals |
| No TTS, no multi-shot assembly | Full presenter pipeline + cinematic assembly |
| Your data in their cloud | Your data on your drive (T9) |

**Sora and Veo are tools. Reel Kingdom Studios is the studio.**

---

## Core Value

1. **Orchestration** — The Render Orchestrator (Agent 5) chooses which shots deserve premium (Sora/Veo) vs. local (ComfyUI). You don’t guess; the pipeline optimizes.

2. **Cost control** — Render all 8 shots locally first. Review. Upgrade only the 2–3 that matter to premium. Set monthly/project caps. Avoid burning credits on every iteration.

3. **Presenter pipeline** — Script → Script Director → Performance Director → reference-image identity → TTS (ElevenLabs) → assembly. Premium APIs don’t offer this end-to-end flow.

4. **Multi-shot production** — Reframing, transitions, background music, trim points, drag-drop reorder. One-click assembly into a final video.

5. **Privacy & ownership** — Storyboards, drafts, and edits stay on your T9 drive. Premium is used only when you choose to ship.

6. **Quality assurance** — Visual QC (frame-based) plus metadata QC. Auto-retry with premium when local fails twice.
